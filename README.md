# Bookary

This is my personal library manager application. It works with Firebase and an external book API to get data based on ISBN13/EAN code.

## Configuration
```js
{
  firebase: {
    apiKey: '',
    authDomain: '',
    projectId: '',
    storageBucket: '',
    messagingSenderId: '',
    appId: '',
  },
  bookApi: {
    url: '',
    version: '',
    key: '',
  },
  email: ''
}
```
